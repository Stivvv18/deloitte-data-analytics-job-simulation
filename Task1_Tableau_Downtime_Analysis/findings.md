# Factory Downtime Analysis

## Objective

Analyze machine telemetry data collected from four Daikibo factories to determine:

1. Which factory experienced the highest machine downtime.
2. Which machine types contributed most to downtime.

## Tools Used

- Tableau
- Data Visualization
- Dashboard Development

## Process

- Imported JSON telemetry data into Tableau.
- Created a calculated field called "Unhealthy" to quantify downtime.
- Built a bar chart showing downtime per factory.
- Built a second bar chart showing downtime per device type.
- Created an interactive dashboard with filtering functionality.

## Results

| Factory | Downtime (Minutes) |
|----------|----------|
| Berlin | 20 |
| Meiyo | 110 |
| Seiko | 480 |
| Shenzhen | 420 |

### Key Finding

Daikibo Factory Seiko recorded the highest downtime with 480 minutes.

Further analysis showed that downtime was concentrated among specific machine types, helping identify operational bottlenecks.
