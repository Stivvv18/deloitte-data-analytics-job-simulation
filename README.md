---

## Task 1: Factory Downtime Analysis

### Objective

Identify:

- Which factory experienced the highest machine downtime
- Which machine types contributed most to downtime

### Tools

- Tableau
- Data Analytics
- Dashboard Design

### Results

| Factory | Downtime (Minutes) |
|----------|----------|
| Berlin | 20 |
| Meiyo | 110 |
| Seiko | 480 |
| Shenzhen | 420 |

### Key Finding

Daikibo Factory Seiko experienced the highest downtime.
